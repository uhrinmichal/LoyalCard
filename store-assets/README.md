# LoyalCard AppGallery assets

This directory contains the source assets and localized copy for the first
AppGallery release.

## Ready

- `loyalcard-icon-v2-master.png`: master app icon
- `store-listing-sk.md`: Slovak store listing
- `store-listing-en.md`: English store listing
- `privacy-policy-sk.md`: Slovak privacy policy draft
- `privacy-policy-en.md`: English privacy policy draft

## Before submission

- Publish the privacy policy at a public HTTPS URL.
- Confirm the final category available in AppGallery Connect.
- Confirm the app is free, contains no ads, and has no in-app purchases.
- Complete the age rating and data collection declarations.
- Select the release countries and regions.
- Capture screenshots using test card numbers only.
- Export the icon to the exact size requested by the submission form.
- Upload a signed release package.

## Screenshot plan

1. Card list with three clearly named test cards.
2. QR card detail with a generated test QR code.
3. EAN-13 card detail with a generated test barcode.
4. Fullscreen scan mode.
5. Card management screen showing rename, edit, First, and delete actions.

Do not use real loyalty card numbers in screenshots.
