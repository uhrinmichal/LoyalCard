# LoyalCard
Loyalty cards application for Huawei Lite Wearable devices.

## Features

- Round-display layout optimized for wearable devices
- QR cards rendered from local image assets
- EAN-13 barcode rendering with fullscreen scan mode
- Demo cards for Lidl, Tesco, Kaufland, and BILLA

## Card data

The cards in this repository use demo values only. Do not commit real loyalty
card numbers or personal QR code assets to the public repository.
