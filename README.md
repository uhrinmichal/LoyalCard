# LoyalCard
Loyalty cards application for Huawei Lite Wearable devices.

## Features

- Round-display layout optimized for wearable devices
- Scrollable list with up to 5 custom cards in any QR/EAN combination
- Add, rename, edit, and delete cards directly on the watch
- Move any card to the first position with persistent ordering
- Touch A-Z and numeric keypads optimized for the round display
- Custom cards restored from wearable system storage with legacy migration
- Native QR rendering and EAN-13 barcode rendering with fullscreen scan mode

## Supported input

- QR cards accept 1 to 32 numeric digits.
- EAN-13 cards accept 12 digits and calculate the check digit, or validate all 13 digits.

## Card data

Card numbers are entered on the watch and stored locally in wearable system
storage. Do not commit real loyalty card numbers to the public repository.
